# Task: Create a Snake Game

Create a browser-playable snake game in HTML/CSS/JS.

## Requirements
- Single HTML file (inline CSS and JS)
- 400x400 canvas, 20x20 grid
- Arrow keys and WASD controls
- Score tracking with localStorage high score
- Start screen, game over screen
- Increasing difficulty (speed) as score increases

## Style Guide

**Aesthetic**: Cyberpunk / Retro Terminal
- Dark background (#050a0e or similar void color)
- Neon green primary (#00ff9f)
- Hot pink/red for danger/food (#ff003c)
- Cyan for UI accents (#00cfff)

**Typography**:
- Use monospace fonts (system or Google Fonts like 'Share Tech Mono', 'Orbitron')
- Glowing text effects with text-shadow
- Uppercase letters with letter-spacing

**Visual Effects**:
- Scanline overlay effect (subtle horizontal lines)
- Glowing borders on the game arena
- Pulsing/animated food items
- Snake head should have eyes that face movement direction
- Gradient body color (brighter head, darker tail)

**Personality**:
- Game title should be thematic (e.g., "SNAKE.EXE", "NEURAL PREDATOR")
- Add a subtitle/tagline
- Game over messages should be dramatic ("SYSTEM FAILURE", "FATAL COLLISION")
- Consider adding bonus items (golden diamonds worth extra points)

**Polish Details**:
- Corner decorations on game arena
- Animated title (flicker or glow pulse)
- Button hover effects
- Pause functionality (P key)

## Quality Bar

The output should feel like a polished mini-game, not a coding exercise.
It should have personality and attention to detail.
Someone should want to play it more than once.
